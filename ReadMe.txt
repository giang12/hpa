This package contains all necessary files to run hpa tool.
	Denpendency:
		-ntlk (wordlink, and tokenizer)
		-standford_parser (chunker)
To run HPA import give_me_all_equiv_phrases([]) from equivalent_sentences_tool
	give_me_all_equiv_phrases([]) takes 1 argument which is a list of 2+ seeding sentences.
	give_me_all_equiv_phrases([]) will return a list of all newly genereted sentences with similar meanning to the seeding sentences.
